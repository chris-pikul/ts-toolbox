  • randomString(length: number, alphaNumeric?: boolean): string

    Generates a string of random length.

    If the alphaNumeric option is true, then only the alpha-numeric characters (0-9, A-Z, a-z) will be used. If false, then any ASCII character can be used. Note, that if not alphaNumeric, spaces will not be included in the character set.

    Parameters

    • length: number

      Length of the string

    • alphaNumeric: boolean = true

      If true, will only be alpha-numeric
