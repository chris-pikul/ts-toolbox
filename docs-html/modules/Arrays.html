arrayEnsureSize

  • arrayEnsureSize(arr: any[], size: number, fillValue?: any): any[]
    • Defined in arrays.ts:68

    Ensures that the input array is a given fixed-length size. The results are a new array regardless of if sizing was needed.

    If the input array is smaller than the desired size then new values are added to the end of the given fillValue value.

    If the input array is larger than the desired size, it is "cropped" to the desired size and returned.

    If the input array is the desired size, then it is cloned and returned.

    throws

    {Error} If the size argument is negative

    Parameters

    • arr: any[]

      Input array

    • size: number

      Desired length

    • fillValue: any = null

    Returns any[]
