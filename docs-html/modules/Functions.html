debounce

  • debounce<CB>(cb: CB, delay: number): WrappedFunctionVoid<CB>
    • Defined in functions.ts:65

    Debounces a function using the given millisecond delay.

    This means, that until the given delay has elapsed with no further calls to this event, nothing will be performed. Once no more calls to this event are performed for the delay duration, then the function will be called.

    Type parameters

    • CB: CallbackFunctionVoid

    Parameters

    • cb: CB

      Callback to execute

    • delay: number

      illiseconds to delay

    Returns WrappedFunctionVoid<CB>

  • throttle<CB>(cb: CB, delay: number): WrappedFunction<CB>
    • Defined in functions.ts:40

    Throttles a function by a given millisecond delay.

    This means that any subsequent calls during this delay are ignored until enough time has elapsed. The first call will be executed as normal.
