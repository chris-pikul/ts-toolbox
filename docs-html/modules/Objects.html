Const objectsAreSameType

  • objectsAreSameType(first: any, second: any): boolean
    • Defined in objects.ts:51

    Attempts to check if the two provided parameters are the same type. For primitives this is a simple typeof check. Objects are a bit trickier.

    For arrays, each entry will be checked against the corrisponding index on the other object. Indices will loop to prevent overflows.

    For objects, the prototypes constructor will be equated to cheat this.

    Parameters

    • first: any

      Any object

    • second: any

      Any object

    Returns boolean

    True if they are the same types
